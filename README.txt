========================================================
  온디맨드 종목 영상 생성 — 독립 저장소용 번들
========================================================

티커를 입력하면 GitHub Actions가 뉴스를 검색·분석해 짧은 영상을 만들고
메일로 보내주는 시스템입니다. 이 폴더를 새 GitHub 저장소에 그대로 올리면 됩니다.

--------------------------------------------------------
[1] 새 GitHub 저장소 만들기
--------------------------------------------------------
1) github.com → 우측 상단 [+] → New repository
2) Repository name: 원하는 이름 (예: stock-brief)
3) Public 또는 Private 아무거나
4) "Add a README file" 체크 → Create repository
   (기본 브랜치는 main 으로 생성됩니다 — 이 번들은 main 기준으로 동작)

--------------------------------------------------------
[2] 파일 업로드 (이 폴더 전체)
--------------------------------------------------------
방법 A) 웹에서 드래그 (가장 쉬움)
  - 새 저장소 페이지 → [Add file] → [Upload files]
  - 이 폴더 안의 항목 전부(.github 폴더 포함)를 끌어다 놓기
  - 아래쪽 [Commit changes] 클릭
  - ★ 업로드 후 저장소에 ".github/workflows/on-demand-video.yml" 이
    보이는지 꼭 확인 (이게 있어야 Actions가 동작)

방법 B) GitHub Desktop 사용
  - 저장소를 Clone → 이 폴더 파일을 복사해 넣기 → Commit → Push

--------------------------------------------------------
[3] GitHub 설정 (한 번만)
--------------------------------------------------------
1) Settings → Secrets and variables → Actions → New repository secret
   아래 값을 등록 (rklb-dashboard 것과 동일하게):
     GEMINI_API_KEY        (필수 — 뉴스 검색·분석)
     ANTHROPIC_API_KEY     (필수 — 영상 대본)
     GMAIL_USER            (메일 발신 주소)
     GMAIL_APP_PASSWORD    (Gmail 앱 비밀번호)
     GMAIL_TO              (받을 메일 주소)
   * YouTube 자동 업로드까지 원하면 GWS_YOUTUBE_TOKEN 등 추가 (선택)

2) Settings → Actions → General → "Allow all actions" 켜져 있는지 확인

3) 리모컨 웹페이지 켜기:
   Settings → Pages → Source: "Deploy from a branch"
   → Branch: main / (root) → Save
   → 1~2분 뒤 주소:  https://<본인아이디>.github.io/<저장소이름>/on-demand.html

--------------------------------------------------------
[4] 사용법
--------------------------------------------------------
- 위 on-demand.html 주소로 접속
- 처음 한 번: "연결 설정"에서 GitHub 토큰(PAT) 저장
    (페이지 안에 발급 방법 안내 있음. 권한은 이 저장소 + Actions: Read and write)
- 티커 입력 (예: TSLA, NVDA / 한국주식은 005930.KS) → [영상 생성 시작]
- 진행 상태가 화면에 표시되고, 완료되면 메일로 영상 도착 (15~30분)

--------------------------------------------------------
참고
--------------------------------------------------------
- 리모컨 페이지는 저장소 주소를 자동 감지하므로 코드 수정이 필요 없습니다.
- config/rules.json 은 옛 RKLB 규칙 파일이지만, 온디맨드는 사용하지 않고
  프로그램 구동에만 필요해 포함했습니다 (그대로 두세요).
- 종목을 처음 요청하면 AI가 회사 정보를 자동 조사해 configs/<티커>/ 에
  저장하고, 다음부터 재사용합니다.
