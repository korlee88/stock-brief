연결 설정 (GitHub 토큰)

✅ 토큰이 이 브라우저에 저장되어 있습니다 (localStorage — 저장소·서버로 전송되지 않음).

토큰 발급 방법 (1분)
  1. GitHub → 새 Fine-grained 토큰 열기
  2. Repository access → Only select repositories → 이 저장소 선택
  3. Permissions → Repository permissions → Actions: Read and write (저장소가 비공개면 Contents: Read-only도 추가)
  4. Generate 후 github_pat_... 값을 여기 붙여넣기

⚠ 토큰은 절대 저장소 파일·코드에 넣지 마세요. 이 페이지는 브라우저 localStorage에만 보관합니다.
